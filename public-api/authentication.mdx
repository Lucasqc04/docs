title: Autenticação

- **Tipo**: Bearer Token
- **Header**: `Authorization: Bearer <token>`
- O token é vinculado a uma loja; os pagamentos criados ficam sempre no contexto desta loja.
- Somente o **token ativo mais recente** da loja é aceito pela API.

## Webhook Secret

Ao criar um novo token de API pública, você recebe também um **webhookSecret** exclusivo. Este secret é usado para assinar digitalmente todos os webhooks enviados pela API, garantindo autenticidade e integridade das notificações.

<Warning>
Guarde o webhookSecret gerado ao criar o token, pois ele não será mostrado novamente.
</Warning>

Para mais detalhes sobre como validar a assinatura dos webhooks, consulte a seção [Webhooks](/public-api/webhooks).

## Como gerar o token

Você pode gerar o token no painel da loja em:

- `https://admin.dominipay.com.br/profile?tab=store`

<Warning>
A geração do token público requer **KYC aprovado** na loja.
</Warning>
