# Endpoints

Explore e teste os principais endpoints da API Pública do PagueBit diretamente abaixo. Utilize o playground interativo para simular requisições e visualizar respostas em tempo real.

---

## Criar pagamento dinâmico

<details>
<summary><b>POST /api-public/payments</b></summary>

<Playground api="POST /api-public/payments" playground="interactive" />

### Parâmetros do corpo (JSON)

<ParamField body="amount" type="number" required>
  Valor entre 5 e 3000 (BRL)
</ParamField>
<ParamField body="email" type="string">
  E-mail do cliente
</ParamField>
<ParamField body="observation" type="string">
  Observação livre
</ParamField>
<ParamField body="webhookUrl" type="string">
  URL para receber webhooks
</ParamField>

### Resposta de sucesso

<ResponseField name="id" type="string" required>
  ID do pagamento criado
</ResponseField>
<ResponseField name="status" type="string" required>
  Status do pagamento (`pending`, `approved`, etc)
</ResponseField>
<ResponseField name="amount" type="number" required>
  Valor do pagamento
</ResponseField>
<ResponseField name="storeId" type="string" required>
  ID da loja
</ResponseField>
<ResponseField name="createdAt" type="string" required>
  Data de criação (ISO)
</ResponseField>
<ResponseField name="webhookUrl" type="string">
  URL do webhook
</ResponseField>
<ResponseField name="observation" type="string">
  Observação livre
</ResponseField>
<ResponseField name="email" type="string">
  E-mail do cliente
</ResponseField>
<ResponseField name="qrCodeUrl" type="string">
  URL do QR Code
</ResponseField>
<ResponseField name="qrCopyPaste" type="string">
  Código copia e cola do Pix
</ResponseField>

#### Exemplo de resposta

```json
{
  "id": "pay_123",
  "status": "pending",
  "amount": 150.0,
  "storeId": "store_abc",
  "createdAt": "2025-12-16T15:00:00.000Z",
  "webhookUrl": "https://example.com/webhook",
  "observation": "Pedido #9876",
  "email": "cliente@example.com",
  "qrCodeUrl": "https://.../qrcode.png",
  "qrCopyPaste": "000201010212..."
}
```

#### Possíveis erros

- `Valor mínimo permitido para API pública é R$ 5,00.`
- `Valor máximo permitido para API pública é R$ 3000,00.`

</details>

## Listar pagamentos por loja

<details>
<summary><b>GET /api-public/payments</b></summary>

<Playground api="GET /api-public/payments" playground="interactive" />

### Parâmetros de query

<ParamField query="page" type="int" default="1">
  Página (≥ 1)
</ParamField>
<ParamField query="limit" type="int" default="50">
  Itens por página (1–200)
</ParamField>
<ParamField query="status" type="string">
  Filtra por `pending`, `approved`, `review`, `not_approved`
</ParamField>

### Resposta de sucesso

<ResponseField name="items" type="array" required>
  Lista de pagamentos
</ResponseField>
<ResponseField name="page" type="int" required>
  Página atual
</ResponseField>
<ResponseField name="limit" type="int" required>
  Limite de itens por página
</ResponseField>
<ResponseField name="total" type="int" required>
  Total de pagamentos encontrados
</ResponseField>

#### Exemplo de resposta

```json
{
  "items": [
    {
      "id": "pay_123",
      "status": "approved",
      "amount": 150.0,
      "storeId": "store_abc",
      "createdAt": "2025-12-16T15:00:00.000Z",
      "webhookUrl": "https://example.com/webhook",
      "observation": "Pedido #9876",
      "email": "cliente@example.com",
      "qrCodeUrl": "https://.../qrcode.png",
      "qrCopyPaste": "000201010212..."
    }
  ],
  "page": 1,
  "limit": 50,
  "total": 237
}
```

</details>
