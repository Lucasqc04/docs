---
title: Criar pagamento
openapi: openapi.json POST /api-public/payments
---

Este endpoint permite criar um pagamento Pix dinâmico e retorna o QR Code correspondente.

Use este endpoint para gerar cobranças Pix para seus clientes de forma automatizada.

<Note>
O saldo só fica disponível quando o status do pagamento for <b>approved</b>.
</Note>
