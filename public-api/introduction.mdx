# API Pública PagueBit

A API Pública PagueBit permite a geração de pagamentos dinâmicos em aplicações externas. Com ela, você pode criar cobranças que ficam disponíveis no dashboard da sua loja e podem ser sacadas a qualquer momento.

Todos os endpoints utilizam o prefixo `/api-public` e exigem autenticação via token de loja.
